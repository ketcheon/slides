Deck Builder

Create lyric and message decks, preview in Reveal.js, and export a standalone HTML file for GitHub Pages.

How Friends Can Use This

Option A: Use Your Hosted Repo

Share your builder URL. They create decks inside your hosted page and export standalone slideshow files when needed.

Option B: Host on Their Own Repo

They copy index.html and slideshow.html into their own GitHub repo, enable Pages, then manage their own decks privately.

Both options are no-backend. For large decks, use Download Standalone HTML instead of long share links.
